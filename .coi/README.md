# Code on Incus (COI)

Follow the [quickstart of COI](https://github.com/mensfeld/code-on-incus/#quick-start)
to install code on Incus.

Then run the following command in the project root directory:

```bash
coi build
```

and then start your first session with:

```bash
coi shell
```

To reattach to the session later, run:

```bash
coi attach
```

or if you want to attach with bash, run:

```bash
coi attach --bash
```
